# Telegram Bot RAG

Бот шукае па PDF-калекцыі беларускіх слоўнікаў і фальклорных тэкстаў. Кожны PDF індэксуецца ў Qdrant з метаданымі `category` і `dictionaryType`, таму бот можа выбіраць асобны тул пад тып пытання.

Архітэктурная прапанова для наступнай версіі RAG-пошуку: [docs/rag-search-architecture.md](docs/rag-search-architecture.md).

## Прыклады Запытаў

### Прыказкі І Прымаўкі

Тул: `folk_wisdom_search`

- `Знайдзі прыказкі пра працу і ляноту`
- `Падбяры некалькі прыказак пра сям'ю`
- `Ці ёсць прымаўкі пра дождж або надвор'е?`
- `Знайдзі народную мудрасць пра розум і дурноту`
- `Пакажы падобныя прыказкі да "Без працы няма чаго есці"`

### Вушацкі Словазбор І Дыялект

Тул: `dialect_dictionary_search`

- `Што ёсць у Вушацкім словазборы пра гразьбы?`
- `Знайдзі праклёны або пагрозы ў Вушацкім словазборы`
- `Што значыць мясцовае слова "..."?`
- `Пашукай дыялектныя словы пра ежу`
- `Пакажы ўстойлівыя выразы з Вушаччыны`

### Кароткія Фальклорныя Запісы Па Катэгорыях

Тул: `record_search`

- `Дай вітанні з Вушацкага словазбору`
- `Пакажы развітанні`
- `Знайдзі добрыя пажаданні`
- `Пакажы праклёны і грозьбы`
- `Знайдзі абразы або цвялілкі`
- `Прыкметы пра дождж`

### Арфаграфічны Слоўнік

Тул: `orthographic_dictionary_search`

- `Як правільна пішацца слова "..."?`
- `Правер напісанне слова "..." па арфаграфічным слоўніку`
- `Ці ёсць форма "..." у слоўніку?`
- `Знайдзі слова "..." у арфаграфічным слоўніку`
- `Як пісаць гэтае слова па-беларуску: "..."?`

### Пераклад

Тул: `translation_dictionary_search`

- `Як па-беларуску будзе "..."?`
- `Перакладзі з расейскай на беларускую: "..."`
- `Як па-руску будзе беларускае слова "..."?`
- `Знайдзі пераклад слова "..." у слоўніку`
- `Ці ёсць у перакладным слоўніку слова "..."?`

### Тлумачальны Слоўнік

Тул: `explanatory_dictionary_search`

- `Што значыць слова "..."?`
- `Растлумач значэнне слова "..."`
- `Знайдзі азначэнне слова "..." у тлумачальным слоўніку`
- `Якое значэнне мае слова "..."?`
- `Пакажы прыклады або тлумачэнне для слова "..."`

### Агульны Пошук Па Калекцыі

Тул: `rag_search`

- `Што ў калекцыі ёсць пра вясну?`
- `Знайдзі ўсе згадкі пра хлеб`
- `Пашукай па ўсіх слоўніках слова "..."`
- `Дзе ў крыніцах сустракаецца выраз "..."?`
- `Падбяры матэрыялы па тэме "народныя прыкметы"`

## Індэксацыя PDF

```bash
bun run index:pdf
```

Індэксуе толькі новыя або змененыя PDF. Стан захоўваецца ў `pdf/.indexed.json`.

```bash
bun run index:pdf --force
```

Прымусова пераіндэксуе ўсе PDF.

```bash
bun run index:pdf --sync-manifest
```

Абнаўляе `pdf/.indexed.json` з ужо існага Qdrant без паўторнай індэксацыі.
